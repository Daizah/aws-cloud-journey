# Host a Website on Amazon S3

## Overview
Deployed a personalized static website to Amazon S3 as part of the 
Nextwork AWS Cloud Beginner series. Configured the bucket for public 
access so the site is accessible to anyone on the internet.

## AWS Services Used
- Amazon S3

## What I Did
- Created an S3 bucket
- Uploaded HTML and image files to the bucket
- Enabled static website hosting on the bucket
- Configured public access settings so the site is publicly accessible

## Key Concepts Learned
- How Amazon S3 stores objects and organizes them in buckets
- The difference between private and public bucket access
- How S3 can serve as a web host without a traditional server
